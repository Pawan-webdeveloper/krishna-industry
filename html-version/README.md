# Krishna Industries — Static HTML Version

A plain HTML, CSS, and JavaScript copy of the React site in the parent project.

## Open locally

```bash
cd html-version
python3 -m http.server 8080
```

Then visit [http://localhost:8080](http://localhost:8080)

Or open `index.html` directly in your browser (images and icons load from local/CDN paths).

## Contents

- `index.html` — full page markup
- `styles.css` — layout, colors, responsive design, animations
- `script.js` — navbar scroll, mobile menu, scroll reveals, stat counters
- `images/` — all 11 site images (hero, products, capabilities, applications, etc.)

## Features replicated

- Sticky navbar with blur on scroll
- Mobile menu toggle
- Hero entrance animation and floating scroll indicator
- Animated stat counters on scroll
- Scroll-reveal section animations
- Product card hover lift and image zoom
- Capability and application image hover effects
- Certification card hover scale
- Contact form slide-in
- Lucide icons (loaded via CDN)
